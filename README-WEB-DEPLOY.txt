HEXA OCTARINA CONQUER — DEPLOY WEB MOBILE
Tehkné Solutions

1. Faça merge do PR de deploy.
2. Acesse https://dashboard.render.com/.
3. Escolha New > Blueprint.
4. Conecte o repositório Tehkne-Solutions/hexa-octarina-conquer.
5. Confirme o arquivo render.yaml.
6. Aguarde o Web Service e o PostgreSQL ficarem verdes.
7. Abra a URL onrender.com no celular.
8. No menu do Chrome, escolha Instalar aplicativo ou Adicionar à tela inicial.

O mesmo domínio fornece a PWA, APIs e o WebSocket /ws.
Não é necessário configurar VITE_HEXA_WS_URL neste modelo.
