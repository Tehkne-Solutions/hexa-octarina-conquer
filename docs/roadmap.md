# Roadmap do projeto

## Fase 1 - Fundação

- criar estrutura do projeto;
- definir o modelo de tabuleiro e províncias;
- implementar regras básicas de fechamento de célula;
- documentar arquitetura e decisões.

## Fase 2 - Evolução estratégica

- adicionar unidades de Go com progressão visual e mecânica de conexão;
- implementar captura por cerco;
- introduzir cartas simples de ataque e evolução.

## Fase 3 - Preparação para experiência viva

- adicionar eventos e logs de turno;
- preparar a base para UI 3D e combate local.
